This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a Python basics course repository for learning Python fundamentals with a focus on AI/ML applications. Lessons are numbered sequentially (`01_lesson.py`, `02_lesson.py`, etc.) and the `docs/` folder contains supplementary HTML reference materials covering topics like git, Python ecosystem, AI/ML paths, and the OSI model.

## Running Code

```bash
python <filename>.py
```

## Structure

- `##_lesson.py` — numbered lesson files added sequentially as the course progresses
- `hello_world.py` — introductory scratch file
- `docs/` — static HTML reference documents (not executed)
